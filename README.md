# hn

## NAME

hn - A Hacker News CLI reader

## INSTALL

`go get github.com/qube81/hn`

## COMMANDS

`hn top [num]`

### EXAMPLE

    hn top // get 10 top stories(default)
    hn top 30 // get 30 top stories(num option max = 500)
